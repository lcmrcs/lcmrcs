## 📌 Featured Project

### [Relatório Executivo de Incidentes Zabbix](https://github.com/lcmrcs/relat.incidentes.zabbix)

A Python-based automation project that connects to the Zabbix API, processes monitoring incidents and generates executive reports in **HTML, Excel and PDF**.

The project helps turn raw monitoring events into a clearer operational view, including:

- incident severity and offline time analysis;
- affected equipment and school unit rankings;
- recurrence and criticality indicators;
- interactive HTML filters and incident details;
- Excel and PDF exports;
- safe publishing workflow with sanitized data and GitHub Actions.
