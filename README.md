## Featured Project

### [Relatório Executivo de Incidentes Zabbix](https://github.com/lcmrcs/relat.incidentes.zabbix)

Python automation that connects to the Zabbix API, processes monitoring incidents and generates executive reports in **HTML, Excel and PDF**.

Key features:

- severity, offline time and recurrence analysis;
- affected unit and equipment rankings;
- interactive HTML report with filters and incident details;
- Excel/PDF exports and sanitized public release workflow.
